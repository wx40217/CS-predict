## 🚀 快速开始指南

### 第一步：自动环境设置（推荐）

**🚀 一键设置环境**：
```bash

python setup_env.py
```

该脚本会自动：
- 检查Python版本和系统要求
- 创建虚拟环境 `cs2_predictor_env`
- 安装所有依赖包（包括CUDA版本的PyTorch）
- 生成激活脚本
- 验证安装

**手动环境设置**（如果自动设置失败）：

1. **检查硬件配置**


```python
    """分析预测置信度"""
    map_probs = list(prediction.map_predictions.values())
    
    # 计算预测的不确定性
    uncertainty = np.std(map_probs)
    
    # 计算最强优势
    max_advantage = max(abs(p - 0.5) for p in map_probs)
```

```python
def objective(trial):
```

## 🔧 故障排除

### 常见问题及解决方案

#### 1. CUDA相关问题

**问题**: `CUDA out of memory`
```bash
# 解决方案1：减少批次大小
# 在config.py中修改
MODEL_CONFIG["batch_size"] = 128

# 解决方案2：启用梯度累积
MODEL_CONFIG["gradient_accumulation_steps"] = 4
```

**问题**: `CUDA not available`
```bash
# 检查CUDA安装
nvidia-smi
nvcc --version

# 重新安装PyTorch CUDA版本
pip uninstall torch torchvision
pip install torch torchvision --index-url https://download.pytorch.org/whl/cu118
```

#### 2. 内存相关问题

**问题**: 系统内存不足
```python
# 解决方案：优化数据加载
```

```python
# 解决方案：数据清洗和验证
```

```python
import matplotlib.pyplot as plt
import seaborn as sns

def analyze_team_performance(team_name, predictions_history):
    """分析队伍历史预测表现"""
    
    # 绘制胜率趋势
    plt.figure(figsize=(12, 8))
    
    plt.subplot(2, 2, 1)
    dates = [p["date"] for p in predictions_history]
    winrates = [p["predicted_winrate"] for p in predictions_history]
    plt.plot(dates, winrates)
    plt.title(f"{team_name} 预测胜率趋势")
    plt.ylabel("预测胜率")
    
    # 地图偏好分析
    plt.subplot(2, 2, 2)
    map_performance = {}
    for pred in predictions_history:
        for map_name, winrate in pred["map_predictions"].items():
            if map_name not in map_performance:
                map_performance[map_name] = []
            map_performance[map_name].append(winrate)
    
    avg_performance = {k: np.mean(v) for k, v in map_performance.items()}
    plt.bar(avg_performance.keys(), avg_performance.values())
    plt.title(f"{team_name} 各地图平均表现")
    plt.ylabel("平均胜率")
    plt.xticks(rotation=45)
    
    plt.tight_layout()
    plt.savefig(f"{team_name}_analysis.png")
    plt.show()
```
